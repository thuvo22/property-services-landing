# OnPoint Pros - Property Services Landing Page

A landing page targeting property managers, landlords, and homeowners who need full-service property maintenance and turnover services.

## Target Audience
- Property Managers (multi-family, apartments)
- Landlords with rental properties
- Homeowners needing renovation/repairs
- Make-ready/Turnover projects

## Services Highlighted
- Interior & Exterior Painting
- Flooring Installation (LVP, laminate, hardwood, tile)
- Tile & Backsplash
- Plumbing Repairs
- Electrical Work
- Drywall Repair & Installation
- Doors, Trim & Carpentry
- General Repairs

## Deployment

### GitHub Pages Setup
1. Create a new repository or use a subdomain
2. Push this folder to the repository
3. Enable GitHub Pages in Settings
4. Add your custom domain via CNAME file

### Custom Domain
Update the `CNAME` file with your desired domain (e.g., `services.onpointprostx.com`)

## Files
- `index.html` - Main landing page
- `CNAME` - Custom domain configuration
- `images/` - Place your images here (logo, before/after photos, etc.)

## Form Integration
The form submits to the same GoHighLevel webhook as the drywall landing page. Update the webhook URL in the JavaScript if needed.

## Customization
- Update the logo path in the header (currently points to drywall-landing-page images)
- Add your own images to the `images/` folder
- Modify pricing in the Pricing Section
- Update testimonials with real customer reviews
